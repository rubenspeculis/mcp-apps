# The Cloudflare tunnel

To test inside a real host (Claude, ChatGPT, VS Code), the MCP server needs a public HTTPS URL. `@mcpapps/dev` ships a Cloudflare tunnel helper.

```bash [Terminal]
pnpm dev --tunnel
```

This starts an ephemeral [Cloudflare quick tunnel](https://developers.cloudflare.com/cloudflare-one/connections/connect-networks/do-more-with-tunnels/trycloudflare/) (`cloudflared`) and prints a public `*.trycloudflare.com` URL. Point a host's MCP connector at `https://<tunnel>/mcp`.

## Requirements

- The `cloudflared` binary: `brew install cloudflared` (or your platform's package).

## Programmatic API

```ts
import { startCloudflareTunnel } from "@mcpapps/dev";

const tunnel = await startCloudflareTunnel({ port: 5179 });
console.log(tunnel.url);   // https://something.trycloudflare.com
// …
tunnel.stop();
```

::note
The tunnel exposes the **same** dev server the emulator uses, so the server, `ui://` resources and `_meta` are identical — you're exercising real host behavior against your local code.
::
