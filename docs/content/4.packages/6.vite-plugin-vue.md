# @mcpapps/vite-plugin-vue

Build-time compilation of Vue SFCs into self-contained `ui://` HTML resources, plus the typed registry that powers Vue inference.

## API

```ts
import { buildVueComponent, writeComponentsModule } from "@mcpapps/vite-plugin-vue";

const component = await buildVueComponent({
  entry: "/abs/path/WeatherCard.vue",
  uri: "ui://weather-app/get_weather",
  root: process.cwd(),
});
// → { uri, html }  (Vue + client-core + the component, inlined into one HTML doc)
```

- `buildVueComponent(options)` → `CompiledComponent` — bundles the SFC, the Vue runtime, and `client-core` into a single self-contained HTML document (CSS and small assets inlined; no external requests).
- `writeComponentsModule(specs, outFile, { root })` — build several components and write a generated TS module exporting one `CompiledComponent` each, which `app.ts` imports.

The `mcpapps` CLI calls `writeComponentsModule` for you on `dev`/`build` from your `mcpapps.config.ts` `components`.
