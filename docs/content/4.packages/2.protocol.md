# @mcpapps/protocol

Pure TypeScript types and constants for the MCP Apps / mcp-ui contract. Zero runtime dependencies — the single source of truth every other package imports.

## Constants

```ts
import { MCP_APP_MIME, UI_SCHEME, HostMethods, AppNotifications } from "@mcpapps/protocol";

MCP_APP_MIME;        // "text/html;profile=mcp-app"
UI_SCHEME;           // "ui://"
HostMethods.CallTool;            // "tools/call"
AppNotifications.ToolResult;     // "ui/notifications/tool-result"
```

Keeping every wire-level string in one place means a spec change is a one-line edit, never a hunt across packages.

## Key types

- `UiMeta` / `ToolMetaWithUi` — the `_meta.ui = { resourceUri }` binding.
- `ToolResultEnvelope<T>` — `{ toolName, structuredContent: T, content?, isError? }`.
- `ThemeState` — `{ colorScheme, tokens }`.
- `CompiledComponent` — `{ uri, html, assets?, basePath? }`, the artifact a renderer produces and the server serves.
- JSON-RPC message types + guards (`isJsonRpcRequest`, `isJsonRpcResponse`, …).
