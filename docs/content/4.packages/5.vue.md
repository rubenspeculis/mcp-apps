# @mcpapps/vue

The Vue renderer: hooks over the host bridge and an adapter that mounts a component inside the iframe.

## Hooks

```ts
import { useToolResult, useCallTool, useTheme } from "@mcpapps/vue";
```

- `useToolResult<"tool">()` → `Ref<output | null>` (replayed on mount, reactive).
- `useCallTool("tool")` → `(args) => Promise<output>`.
- `useTheme()` → `Ref<ThemeState>`.

## Typed registry

```ts
declare module "@mcpapps/vue" {
  interface ToolRegistry {
    get_weather: { input: { city: string }; output: { tempC: number } };
  }
}
```

## Adapter

`createVueAdapter(rootComponent)` returns a `RendererAdapter` used by `mountApp` — the Vite plugin generates the bootstrap that calls it, so you rarely touch it directly.

See the [Vue guide](/guides/vue-renderer).
